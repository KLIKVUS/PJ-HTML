Теги:
    <em></em>  -  выделяет курсивом, альтернатива тег <i></i>
    <strong></strong>  -  делает текст жирным
    <del></del>  -  зачеркивает текст
    <ins></ins>  -  визуально подчеркивает

    <sub></sub>  -  сдвигает текст вниз
    <sup></sup>  -  сдвигает текст вверх

    <cite></cite>  -  для помечения цитат

    <span></span>  -  невидемый тег

    <a></a>  -  тег нужный для создания кнопок или ссылкок в текте, если прописать href = "cылка" , также если прописать target = "_blnak" то ссылка откроется в новой вкладке
    
    <ul></ul> - неупордоченный список
    <ol></ol> - нумерация
    	<li></li> - лист

    <style></style> - создает стили при помощи которых можно изменять визуальную часть сайта
        font-size - задает размер текста
        font-indent - создает красную строку и задает ее размер

    Команды:
    p*7>lorem200  -  делает 7 абзацев из 200 слов
    (h1>Изображения)+((h2>lorem10)+(p*3>lorem70))*2+(p*2>lorem70) - можно быстро создавать несколько обзацев и заголоваков

    Основные цвета ( RGB ):
        ( 0 0 0 )  -  black
        ( 255 255 255 )  -  whit
        ( 0 255 255 )  -  aqua / cyan
        ( 255 0 255 )  -  magenta
        ( 255 255 0 )  -  yellow

    body {
          color: rgb(54, 54, 54); 
          background-color: #7C9EA6;      /* Задает цвет заднего фона сайта */
          background-image: url("../images/BgAnim.gif"); background-size: 2450px;     /* Вставляем картинку на сайт, cover - картинка заполняет все пространство */
          background-position: center;      /* Позиция картинки по оси X */
          background-repeat: no-repeat; /* Повторять по оси X repeat-x , Повторять по оси Y repeat-y */
          background-attachment: fixed; /* Закрепляет картинку в одном месте */
    }
